# Фич-роадмап «Распевка» (приоритеты по фич-гэпам конкурентов)

_Создано 2026-06-04. Рамка: приложение = УДЕРЖАНИЕ + воронка в школу «Прояви», не главный заработок._
_Почти всё дешевеет, если сначала собрать маскота Кваковского — он «носитель» (план дня, теория, дуэты, празднования)._

## Адаптация под устройства (phone/tablet/desktop)
- Сейчас: mobile-first, колонка max-width 440px, canvas ресайзится — работает везде, но тратит место на широких экранах.
- **Решение: отложить реальную tablet/desktop-раскладку до этапа «во что оборачиваем»** (TWA/Capacitor/desktop-PWA диктуют нюансы; это полировка, не архитектура).
- **Сейчас соблюдать принцип:** верстать в rem/%/dvh, не в фиксированных px, не завязываться на 440px жёстко → позже адаптация = пара брейкпоинтов (крупнее хайвей + контролы сбоку на десктопе, 2 колонки на доме), а не переписывание.

## Приоритеты фич (что делаем)

### Tier 1 — дёшево + макс. удержание (частично уже есть)
1. **Дашборд + история диапазона (#1)** — данные (rangeHistory) и базовый дашборд УЖЕ есть. Довести: график диапазона во времени, бесты по упражнениям, тренд точности, **празднование «новая нота!»**. ~3-6 ч. #1 против оттока. Первым после маскота.
2. **«Сегодня» — план дня + стрик (#5)** — одна кнопка «тренировка на сегодня» вместо меню. План дёшев, сильно держит. Ведёт Кваковский. **Push — отложить до обёртки** (Capacitor = нативные уведомления легко; PWA-push на iOS слабый).
3. **Микро-уроки теории карточками (#6)** — дёшево (текст), премиум-ценность, идеальная работа Кваковского. В связке с маскотом.

### Tier 2 — среднее усилие, сильный стратегический фит
4. **Ear-training: интервалы/сольфеджио (#4)** — на нашем движке (питч+поводырь). **Это и есть дуэты Кваковского** (поёт ноту → повторяешь). Сливается с маскотом. Новая аудитория. ~2-4 д.
5. **Структурированный путь/курс (#2)** — статический граф уроков, разблокировка по точности. Скелет, что организует всё + **якорь подписки** (free=первые, paid=весь путь). ~1-2 д.

### Tier 3 — позже / лёгкая версия
6. **Песни — лёгкая версия (#7)** — НЕ лицензионная библиотека (дорого/бэкенд), а **5-10 своих royalty-free мелодий**, скоринг как в хайвее. Главный крючок привлечения+шеринга. Когда ядро+маскот готовы.
7. **Шеринг-карточка результата/диапазона (часть #9)** — ~1 д, без бэкенда, вирал → кормит маркетинг.

## Откладываем / пропускаем
- **Анализ вибрато/придыхания (#3)** — дорогой DSP, узкий сегмент сейчас. Дифференциатор, но позже.
- **Офлайн / service worker (#10)** — почти всё уже офлайн (Web Audio); делаем на этапе обёртки (TWA/Capacitor всё равно требуют SW).
- **Полноценная соц-сеть / лидерборд (#9)** — нужен бэкенд, риск «пустого» лидерборда. НЕ делаем (только шеринг-карточка).
- **Конструктор гамм (#8)** — нишево; приберечь как B2B-фишку для школы (педагоги задают гаммы ученикам).

## Рекомендованный порядок сборки
Кваковский → дашборд+«новая нота» → «Сегодня»+план → теория-карточки → структурированный путь → ear-training/дуэты → лёгкие песни → шеринг-карточка.
(Вибрато-анализ, офлайн, push, конструктор гамм — отдельным горизонтом / на этапе обёртки.)
