# Сверочный лист для музыканта — ноты распевок (L02–L10)

Ноты сняты **программно** из PDF (по координатам нотных голов — не на глаз).
В приложении всё **транспонируется под голос ученика**; здесь — исходные высоты из PDF.
Проверь, пожалуйста, каждую строку: ✅ если совпадает с оригиналом, ✍ если поправить.

## Скачок к V (по твоей правке)
Тоника … **квинта ВНИЗ** … возврат: I-I-III-II-I-I-**V(низ)**-I-I-III-II-I. Подтверди, что так.

## Как проверить вживую
1. Открой https://a1exxx.github.io/raspevka/ (перезагрузи пару раз — подтянется свежее).
2. «Свободная практика» → категория → распевка. На экране объяснения виден **рисунок мелодии**, при старте играет **образец** — слушай его.
3. Если где-то не так: скажи номер/название и что не так («выше/ниже на …», «не та фигура») — поправим за минуты, ноты лежат структурированно.

_Темпы и слоги расставлены по смыслу упражнений — тоже глянь, если будет время._
