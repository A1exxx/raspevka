1Продукт и технологии

  • Приложение «Распевка»: web-first PWA (vanilla JS + Vite + Canvas), детекция высоты тона в реальном времени (pitchy/MPM), без бэкенда — всё в localStorage. Нулевая инфра-стоимость, мгновенный деплой. Live: a1exxx.github.io/raspevka/
  • Школа «Прояви»: онлайн-уроки + лендинг + CRM. Главный денежный поток на старте.
  • Маскот «Кваковский» (вектор-SVG-риг) собран, временно отключён (код в src/ui/mascot.js, тег approved-2026-06-04-mascot-kvakovsky).

2Публикация приложения

  • Google Play: обёртка PWA через Bubblewrap (TWA) или PWABuilder. $25 разово. Нюансы: микрофон RECORD_AUDIO дописывать патчем, Digital Asset Links, target API 35, Data Safety + Privacy Policy. Риск «web-wrapper» → нужны нативные плюсы.
  • iOS: чистый PWA не пройдёт (4.2) → Capacitor + 2–3 нативных элемента, $99/год, NSMicrophoneUsageDescription.
  • RuStore: через Capacitor (TWA не поддерживается), комиссия 15%, оплата работает в РФ.

3Монетизация / биллинг (РФ-реалии)

Google Play Billing для РФ-разработчиков отключён с дек 2024

Деньги — через веб-подписку: ЮKassa / CloudPayments (МИР/СБП/рекуррент/54-ФЗ, комиссия ~3%) + RuStore Pay (15%) + Telegram Stars. В магазинах — бесплатное приложение со ссылкой на веб-оплату.

Цена приложения: 690 ₽/мес или 3 990 ₽/год; школа — абонемент 18 000 ₽/мес, педагогу 40–50%.

5Воронка и перелив app ↔ школа

Короткое видео (VK Клипы / Shorts / Telegram) → общий Telegram-бот (app + школа) → квиз «какой у тебя голос» (лид-магнит) → бесплатный пробный урок школы + «качай приложение» → абонемент (20–35% от пришедших на пробный) → приложение удерживает между уроками

Прямой мост «free app → дорогой курс» НЕ работает без промежуточного шага (Babbel/JoyTunes сворачивали). Рабочее: app = habit-движок, школа = отдельная воронка, пересекаются через триггеры (in-app CTA по intent, companion-офферы, общий бот, контент педагога). Синергия → CAC −20–30%.

6Юнит-экономика — до $20k/мес (≈1,85 млн ₽)

Три пути к одной цели — отличаются долей выручки школа/приложение. A: ~80% школа (быстро, маржа ниже — платим педагогам). C «Баланс»: ~50/50 школа/приложение. B: ~80% приложение (маржа выше, но дольше набирать аудиторию).

7Контент-завод (Claude Code + Higgsfield)

  • Льём в VK Клипы / Shorts / Telegram (НЕ Instagram-таргет РФ).
  • Форматы: до/после голоса, «5 ошибок», «упражнение за 60 сек», страх сцены, каверы.
  • Анти-бан: каждый ролик — уникальный угол (не шаблон-клоны — кейс @deadline.dev); 2 аккаунта = 2 стиля; прогрев 14 дней; ~1 ролик/день; только официальные API.

8Продуктовый роадмап

Кваковский (вернуть) → дашборд + «новая нота» → «Сегодня» + план → теория-карточки → структурированный путь (якорь подписки) → ear-training/дуэты → лёгкие песни → шеринг-карточка.

Монетизация: free = ядро + первые уроки; paid = весь путь + ear-training + песни. Адаптация устройств — на этапе обёртки; сейчас верстать гибко (rem/%/dvh).

Подробно → feature-roadmap.md

9Риски

  • Instagram/РФ-легал — обходим каналами (VK/Telegram/Яндекс).
  • Чёрн школы (вокал-хобби 10–15%/мес) — держим контентом + дашбордом прогресса + приложением.
  • App-конверсия (freemium 3–5%) — зависит от onboarding + paywall по intent.
  • Платёжная инфра РФ — ЮKassa/CloudPayments как основа.

10Ближайшие шаги

  • Включить веб-оплату приложения (триал 7 дн → 690 ₽/мес / 3990 год) через ЮKassa.
  • Поднять школу как поток №1: лендинг + квиз + общий Telegram-бот + пробный урок.
  • Запустить VK + Яндекс Директ (тест 100–150к ₽/мес), контент-завод в VK Клипы/Shorts.
  • Достроить приложение по роадмапу (дашборд → план → теория → путь).
  • Обернуть в Google Play (Bubblewrap) + RuStore (Capacitor); iOS — по решению.
