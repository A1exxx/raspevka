# Маркетинг-стратегия: «Распевка» (приложение) + «Прояви» (школа) → $20k/мес

_Создано 2026-06-04. На основе ресёрча (4 агента, источники с датами внутри). Курс ~92 ₽/$ → цель ≈ 1,85 млн ₽/мес._

---

## 0. КРИТИЧЕСКИЙ reality-check: Instagram для РФ — НЕЛЬЗЯ как канал

Исходный план (2 инсты + таргет + контент-завод рилсов) для РФ-аудитории **юридически запрещён**:
- Meta признана экстремистской (2022), Instagram заблокирован (только VPN).
- **Закон 72-ФЗ с 01.09.2025: реклама НА ресурсах Meta запрещена.** Сюда входят: таргет, платные посты у блогеров, **бартерные** интеграции, продающий контент с ценами/промокодами/«ссылка в bio».
- Штрафы по ст. 14.3 КоАП: **юрлицо 100 000–500 000 ₽ за КАЖДЫЙ пост/сторис/рилс**. Применяется ретроактивно к старому контенту.
- Оплата Meta Verified квалифицирована судом (авг 2025) как финансирование экстремизма — ст. 282.3 УК, до 8 лет. **Никаких платежей в адрес Meta.**

**Вывод:** Instagram оставляем только как **органический информационный** аккаунт (без оплаты, без продающих CTA) под глобал/русскоязычную диаспору. Весь РФ-трафик и деньги — на **VK, Telegram, Яндекс**. Контент-завод (Claude Code + Higgsfield) остаётся, но льёт в **VK Клипы / YouTube Shorts / Telegram**, не в Instagram-таргет.

Источники: Moscow Times 07.04.2025, Octo Browser, RBC Компании (72-ФЗ), Moscow Times 22.08.2025.

## 2. Воронка (реальные конверсии РФ)

```
Короткое видео (VK Клипы/Shorts/Telegram) 
 → подписка на канал/общий Telegram-бот (app + школа в одном боте)
 → ЛИД-МАГНИТ: квиз «Какой у тебя голос/тип?» (2-3 вопроса) [квиз > простой формы]
 → результат + оффер БЕСПЛАТНОГО пробного урока школы
 → параллельно: «качай приложение для ежедневной практики»
 → пробный урок (приход 60-70% записавшихся)
 → покупка пакета (20-35% от пришедших на пробный)
 → приложение удерживает между уроками + push/email с оффером
```
**Сквозная клик→платящий клиент школы: ~1-3%** (до 5% при квизе + SMS-напоминаниях + разборе в 24ч).
**CAC школы:** при лиде 200-500 ₽ и конв. лид→ученик ~9-30% → **1 800–5 000 ₽**.
**App:** freemium→платно **3-5%** (геймификация может дать 5-7%), триал 7 дн → **30-45%**.

Ключ к 10% вместо 2%: квиз вместо формы, time-limited триал, paywall по «intent» (после N сессий), разбор педагога в 24ч после пробного.
Источники: iSpring, CarrotQuest, lpgenerator, adapty.io, callin.io (пробный урок → запись ×4.3).

## 3. Цены и юнит-экономика

**Школа «Прояви» (онлайн):**
- Индивидуальный урок 60 мин: **2 500 ₽** (регион/онлайн 1500-2500; Москва-топ до 4200).
- Абонемент 8 уроков/мес: **~18 000 ₽** (скидка 12-15%). Группа: 1 200 ₽/чел × 4.
- Пробный: бесплатно или 700-1500 ₽.
- **Выплата педагогу 40-50%** от урока → маржа школы до прочих расходов **50-60%**.
- Чёрн: вокал-хобби **10-15%/мес**; LTV при 6-10 мес ≈ **200 000-270 000 ₽**.

**Приложение «Распевка»:**
- РФ-цена: **690 ₽/мес** или **3 990 ₽/год** (≈333 ₽/мес), blended ARPU ~420 ₽/мес. (конкуренты РФ: Vocal Image 899 ₽/мес, Voca Stock от 399 ₽/мес).
- Чёрн: годовые −7%/мес, месячные −14%/мес (годовые держат ×7 лучше). 30% годовых отменяют в 1-й месяц.

**Прочие расходы/мес:** менеджер продаж 80-120к, админ/куратор 50-70к, CRM 3-15к, эквайринг ЮKassa **~4%**, налог УСН **6%**, инфра приложения 18-90к.
Источники: Mezzo.pro, Artvocal, Profi.ru, repetitor.ru, RevenueCat 2025, ЮKassa.

## 5. Перелив трафика app ↔ школа (синергия)

**Главный вывод ресёрча:** прямой мост «скачай бесплатное приложение → купи дорогой курс» работает ПЛОХО без промежуточного шага (Babbel/JoyTunes сворачивали такой upsell). Лучший паттерн — **app = ежедневный habit-движок, школа = отдельная воронка, пересекаются через триггеры:**
1. **Общий Telegram-бот** для обоих брендов (один контакт, кросс-пуши). Дёшево, управляемо.
2. **In-app CTA по intent:** после N сессий — «прошёл 10 распевок — хочешь живой разбор с педагогом?» → пробный урок школы.
3. **Companion-оффер:** app говорит «результат ×2 с живым педагогом раз в неделю»; школа — «практикуй между уроками в приложении». Взаимоусиление без рекламы.
4. **Контент педагога:** педагог «Прояви» снимает разбор упражнений из «Распевки» → контент для обоих аккаунтов.
5. **Совместный лид-магнит:** «7-дневный план: утром упражнение в app, вечером мини-урок школы».

Синергия снижает общий CAC на **20-30%** (студенты школы = органический трафик в app и наоборот).
Питфолы: НЕ copy-paste между аккаунтами (Instagram non-recommendable за дубли); ротация месседжей (аудитории пересекаются); кросс-промо подавать как «от команды создателей», не как накрутку.
Источники: Smule (дуэты −20-35% CAC), Duolingo (in-app upsell, не cross-sell), Ladder, Fender Play, Babbel, vc.ru автоворонки.

## 6. Контент-завод (Claude Code + Higgsfield) — как делать БЕЗОПАСНО

Льём в **VK Клипы + YouTube Shorts + Telegram** (НЕ в Instagram-таргет для РФ).
Рабочие форматы (из 25 сценариев): **до/после голоса**, «5 ошибок» (миф-бастинг), «упражнение за 60 сек» (делай со мной), демонстрация высокой ноты + механика, страх сцены/зажим, каверы в разных стилях, флешмоб-вызов (UGC). Хук в первые 3 сек (вопрос/контраст/триггер действия). CTA → общий Telegram-бот.

Правила анти-бана (даже для VK/Shorts и особенно если ведём Instagram органикой):
- **Каждый ролик — уникальный угол + уникальное оформление.** Никаких шаблон-клонов (кейс @deadline.dev: ~10 клонов/день → non-recommendable + блок монетизации).
- Два аккаунта = два визуально разных стиля, разные голоса/музыка.
- **Прогрев нового аккаунта 14 дней** до коммерции; ссылку в bio не с 1-го дня.
- Постинг ~1 ролик/день на аккаунт (не 3+), интервал ≥6 ч.
- AI-контент: помечать (где требуется), стилизованный арт безопаснее реалистичных дипфейков.
- Автопостинг — только официальные API/планировщики, не серые боты.
Источники: Tubefilter/PetaPixel (Instagram 30.04.2026 original-content), Metricool лимиты, 360uniquizer прогрев, vechkasov/raish сценарии.

## 7. Что делаем (приоритеты)

1. Включить монетизацию «Распевки» (триал 7 дн → 690 ₽/мес или 3990/год) через веб-оплату ЮKassa.
2. Поднять школу «Прояви» как главный денежный поток: лендинг + квиз-лид-магнит + общий Telegram-бот + пробный урок.
3. РФ-трафик: VK Реклама (школа лид 200-520 ₽) + Яндекс Директ Поиск + Telegram-посевы. Бюджет теста 100-150к ₽/мес.
4. Контент-завод → VK Клипы/Shorts/Telegram, уникальные ролики, CTA в бот.
5. Перелив app↔школа через бот + in-app CTA по intent + companion-офферы.
6. Instagram — только органика под глобал/диаспору, без оплаты и продающих CTA (72-ФЗ).
7. Цель: сценарий A (школа) → C (баланс), 8-18 мес, стартовый бюджет ~500к-1,5млн ₽.
