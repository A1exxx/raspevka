# Go-to-Market: «Распевка» — приложение, публикация, монетизация, конкуренты

_Создано 2026-06-04. На основе ресёрча (источники с датами внизу разделов). Решения по продукту — отдельно._

## 1. Как сделать «настоящим приложением» и опубликовать

### Android — Google Play
Два пути обернуть наш Vite/PWA:
- **TWA (Trusted Web Activity)** через **Bubblewrap CLI** (Google) или **PWABuilder** (Microsoft, GUI). Сайт рендерится в Chrome без адресной строки.
  - Требования: HTTPS, manifest (иконка 512×512), service worker с offline-fallback, **Lighthouse PWA ≥ 80**, Digital Asset Links (`/.well-known/assetlinks.json` с SHA-256).
  - **Подводный камень с микрофоном:** Bubblewrap перезаписывает `AndroidManifest.xml` при каждом build → `RECORD_AUDIO` надо дописывать патч-скриптом ([bubblewrap issue #724]).
  - **Asset Links дважды:** свой ключ при разработке + Play App Signing key после публикации.
  - **Target API:** новые приложения с 31 авг 2025 — **API 35 (Android 15)**.
- **Capacitor** (Ionic) — оборачивает в нативный WebView, **полный контроль над манифестом** (RECORD_AUDIO просто), нативные плагины (haptics, push). Нужен, если будет **iOS** или **RuStore** (TWA там не работает).

Play Console: **$25 единоразово**, формат `.aab`, обязательны Privacy Policy URL + Data Safety (декларировать микрофон) + Content Rating. Ревью обычно ~24 ч. **Риск отказа:** «low-value web wrapper» — нужны app-преимущества (offline, нативные уведомления, спец-UX).

### iOS — App Store
- **Чистый PWA-wrapper НЕ пройдёт** (Guideline 4.2 «minimum functionality»).
- Реальный путь — **Capacitor** + 2-3 нативных элемента (haptics на попадание, нативный splash, обработка прерываний/наушников, offline). `getUserMedia` в WKWebView работает с iOS 14.3+ (hostname localhost). `NSMicrophoneUsageDescription` обязателен. **$99/год.**

### Рекомендованный порядок (соло RU-разработчик)
1. **PWA + «на главный экран»** (0₽, уже почти готово — добить service worker/offline).
2. **Google Play через Bubblewrap** ($25), приложение **бесплатное**, монетизация через сайт.
3. **RuStore** через Capacitor (TWA не поддерживается; комиссия 15%, ревью ~1 ч, оплата работает в РФ).
4. **iOS через Capacitor** ($99/год) — если решим выходить.

Источники: developer.android.com (TWA, target API), github.com/GoogleChromeLabs/bubblewrap (+issue #724), pwabuilder-google-play docs, capacitorjs.com, mobiloud.com (App Store 4.2), rustore.ru/developer — все 2025-2026.

## 2. Монетизация (как у конкурентов)

**Доминирующая модель: freemium + ГОДОВАЯ подписка** (≈80% рынка). Hard-paywall редок; one-time — только утилиты.

**Ценовые полосы (глобал):**
- Entry: $0.99–4.99/мес (Pocket Pitch, Vanido)
- Mid (центр тяжести): **$7.99–14.99/мес** (Vocalista, Riyaz, Erol)
- Premium: $19.99–29.99/мес (Yousician Premium+, Simply Sing, Sing Sharp)
- **Годовая «сладкая точка»: $40–90/год** (= $3.5–7.5/мес после скидки)
- **Триал: 7 дней** — стандарт (Yousician, Vocalista, Vanido).

**Бенчмарки (RevenueCat State of Subscription Apps 2025):**
- Конверсия freemium→платно (D35): **медиана 2.18%** (hard paywall ~12%).
- Триал→платно: **~35–45%** (длиннее триал → выше).
- Education: годовая подписка ~$44 медиана; LTV года ~$8 (дешёвые) … ~$55 (дорогие) на платящего.
- **Годовая >> месячной** по удержанию (≈70% доживают до 12 мес против 17% у месячной).

**Путь к $10k/год (математика):**
- При **$40/год и 2% конверсии** нужно **~12 500 установок** на $10k ARR.
- При **$89/год** — ~5 600 установок.
- **7-дневный триал поднимает конверсию в 5-10×** → достаточно ~1.5–3k установок.
- Вывод: **годовая ≥ $40/год + 7-дневный триал.** Для РФ — отдельная (более низкая) цена.

### Биллинг — критично для РФ
- **Google Play Billing для РФ-разработчиков ОТКЛЮЧЁН с 26 дек 2024, бессрочно.** Платные приложения/IAP/подписки не работают.
- **Решение:** продавать подписку **через сайт** (PWA на HTTPS) — **ЮKassa (Сбер)** или **CloudPayments (Тинькофф)**: МИР, СБП, рекуррентные, чеки 54-ФЗ, комиссия ~3% (вместо 15-30%!). Плюс **RuStore Pay SDK** (15%) для RuStore-юзеров. Плюс **Telegram Stars** для TG-аудитории.
- В США с 2025 (решения Epic): Google и Apple не могут брать комиссию с внешних платежей — для глобал можно вести оплату на сайт.

**Рекомендованный стек монетизации:** основа — **веб-подписка (ЮKassa/CloudPayments, ~3%)**; для RuStore — их Pay SDK; Google Play/App Store — бесплатное приложение со ссылкой на веб-оплату; опционально Telegram Mini App + Stars.

Источники: revenuecat.com/state-of-subscription-apps-2025, support.google.com (suspension RU answer 15685001), yookassa.ru, rustore.ru, core.telegram.org/bots/payments-stars — 2025-2026.

## 4. Чем мы УЖЕ лучше / уникальны
- **Дыхательный модуль** (box/живот/долгий выдох, формат «объяснение→интерактив») — у 11 рассмотренных конкурентов аналога нет.
- **Рисунок-контур мелодии** на карточках + выбор тембра поводыря — не встречаются у других.
- **Свободный режим с нотной шкалой** («Распевайся») — нагляднее, чем у SWIFTSCALES.
- Эти фичи — наша подпись, сохраняем.
