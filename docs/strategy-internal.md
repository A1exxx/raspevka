## 1. Продукт и технологии
- **Приложение «Распевка»:** web-first PWA (vanilla JS + Vite + Canvas), детекция высоты тона в реальном времени (pitchy/MPM), без бэкенда — всё в localStorage. Live: https://a1exxx.github.io/raspevka/. Это даёт нулевую инфра-стоимость на старте и мгновенный деплой.
- **Школа «Прояви»:** онлайн-уроки (Zoom/аналог) + лендинг + CRM. Главный денежный поток на старте.
- **Маскот «Кваковский»** (вектор-SVG-риг) — собран, сейчас отключён в UI (код в `src/ui/mascot.js`, тег `approved-2026-06-04-mascot-kvakovsky`), вернёмся.

## 2. Публикация приложения (как «настоящее»)
- **Android / Google Play:** обёртка PWA через **Bubblewrap (TWA)** или PWABuilder. $25 разово. Нюансы: микрофон `RECORD_AUDIO` дописывать патчем (Bubblewrap затирает манифест), Digital Asset Links, target API 35, Data Safety + Privacy Policy. Риск «web-wrapper» → нужны нативные плюсы.
- **iOS / App Store:** чистый PWA не пройдёт (4.2) → **Capacitor** + 2-3 нативных элемента, $99/год, `NSMicrophoneUsageDescription`.
- **RuStore:** через Capacitor (TWA не поддерживается), комиссия 15%, оплата работает в РФ.
- Подробно → `go-to-market.md`.

## 3. Монетизация / биллинг (тех-реалии РФ)
- **Google Play Billing для РФ-разработчиков отключён с дек 2024.** → деньги через **веб-подписку**: ЮKassa / CloudPayments (МИР/СБП/рекуррент/54-ФЗ, комиссия ~3%). + RuStore Pay (15%), + Telegram Stars.
- В магазинах публикуем **бесплатное** приложение со ссылкой на веб-оплату.
- Цена приложения: **690 ₽/мес или 3 990 ₽/год**; школа: абонемент **18 000 ₽/мес**, педагогу 40-50%.

## 4. Каналы (РФ-легальные) и цифры
- **VK Реклама** — основной (лид школы 190-520 ₽ по кейсам; CPI приложения 35-89 ₽).
- **Яндекс Директ Поиск** — горячий спрос (CPL 400-1000 ₽, конв. лид→клиент до 32%).
- **Telegram** (МТС Маркетолог CPM от 75 ₽; посевы 1-8к ₽/пост).
- **VK Клипы / YouTube Shorts** — органика 0 ₽ для контент-завода.
- ⚠️ **Instagram для РФ — НЕЛЬЗЯ** (Meta экстремист; 72-ФЗ с 01.09.2025 запрещает рекламу НА инсте, штраф 100-500к ₽/пост; оплата Meta Verified = ст. 282.3 УК). Только органика под глобал/диаспору, без оплаты и продающих CTA.
- Подробно → `marketing-strategy.md`.

## 5. Воронка и перелив app↔школа
```
Короткое видео (VK Клипы/Shorts/Telegram)
 → общий Telegram-бот (app + школа)
 → квиз «какой у тебя голос» (лид-магнит)
 → бесплатный пробный урок школы + «качай приложение»
 → абонемент (20-35% от пришедших на пробный); приложение удерживает между уроками
```
- Прямой мост «free app → дорогой курс» НЕ работает без промежуточного шага (Babbel/JoyTunes сворачивали). Рабочее: **app = habit-движок, школа = отдельная воронка, пересекаются через триггеры** (in-app CTA по intent, companion-офферы, общий бот, контент педагога).
- Синергия → CAC −20-30%.

## 7. Контент-завод (Claude Code + Higgsfield)
- Льём в **VK Клипы / Shorts / Telegram** (НЕ Instagram-таргет РФ).
- Форматы: до/после голоса, «5 ошибок», «упражнение за 60 сек», страх сцены, каверы.
- Анти-бан: **каждый ролик — уникальный угол** (не шаблон-клоны — кейс @deadline.dev), 2 аккаунта = 2 стиля, прогрев 14 дней, ~1 ролик/день, только официальные API.

## 8. Продуктовый роадмап приложения (удержание + воронка)
Кваковский (вернуть) → дашборд+«новая нота» → «Сегодня»+план → теория-карточки → структурированный путь (якорь подписки) → ear-training/дуэты → лёгкие песни → шеринг-карточка. Монетизация: free = ядро+первые уроки; paid = весь путь + ear-training + песни. Подробно → `feature-roadmap.md`.
- Адаптация устройств (phone/tablet/desktop) — на этапе обёртки; сейчас верстать гибко (rem/%/dvh).
