# Безопасность голоса

Эти правила зашиты в приложение (экран-дисклеймер перед первой распевкой) и должны
соблюдаться всегда. Источники — ниже.

## Правила
1. **Боль = стоп.** Любой дискомфорт или боль в горле → немедленно прекратить, дать голосу отдых.
2. **Разогрев обязателен.** Никогда не начинать с гамм/прыжков/беглости — только после мычания и сирен.
3. **Не форсировать верхние ноты.** Расширение диапазона — процесс недель/месяцев, не одного урока.
4. **Не петь на больном горле.** Воспаление → связки отекают → риск узелков при форсировании.
5. **Гидратация.** Достаточно воды; избегать кофеин/алкоголь перед пением.
6. **Лимит на старте** — не дольше 20–30 минут подряд без перерыва.
7. **Приложение не заменяет педагога/фониатра.** При проблемах с голосом — к специалисту.
