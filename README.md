# Распевка — игровой вокальный тренажёр

Веб-приложение (PWA): пой в микрофон, приложение слышит высоту голоса в реальном
времени и ведёт по распевкам и упражнениям в игровой форме. Вдохновлено Vocalista
(«Sing on Mah»), но web-first, русскоязычное и с упором на корректную методику.

## Запуск
```bash
npm install
npm run dev      # http://localhost:5173
npm test         # smoke-тесты математики (ноты/центы/зоны/скоринг)
npm run build    # прод-сборка в dist/
```
Тест с телефона требует HTTPS (микрофон) — см. `docs/DEV.md` (ngrok/mkcert).

## Что есть (MVP, Фазы 1–4)
- **Детекция тона** в реальном времени: pitchy (MPM) + AnalyserNode, EMA-сглаживание, iOS-safe AudioContext.
- **Живой тюнер**: нота, центы, Гц, чистота, скользящий след тона.
- **Нотный хайвей** (Canvas): целевые ноты едут к линии попадания, голос — светящийся шарик; зоны ±20/40 центов.
- **Скоринг**: % времени в зелёной зоне → звёзды (1–3), счёт попавших нот.
- **Онбординг диапазона**: ловим комфортные низ/верх; упражнения подстраиваются.
- **Полная распевка**: 5 упражнений в методически верном порядке (разогрев→гибкость→интонация→беглость→охлаждение).
- **Стрик** по дням (localStorage), эталонные тоны (OscillatorNode), дисклеймер здоровья голоса.
- **Дыхательные упражнения** (формат «короткое объяснение → интерактив»): дыхание по квадрату (4-4-4-4 с дышащим кругом), дыхание животом (4-6), долгий выдох «с-с-с» (замер ровного выдоха по громкости + рекорд).
- **Тип голоса**: выбрать вручную (бас…сопрано) или определить — спеть низ и верх (с усреднением и проверкой стабильности) → классификация по диапазону. Упражнения берут корень из центра голоса.
- **Уровни сложности** (темп: медленно/средне/быстро) + **звук-поводырь** (тихо подыгрывает нужную ноту, чтобы попадать ухом) + устойчивая детекция: кламп по диапазону голоса (режет октавные ошибки/шум) и компенсация задержки в скоринге.
- **Объяснение перед каждым упражнением** (что тренирует + как делать + как устроена игра).

## Формат упражнений (паттерн)
Каждое упражнение = **короткое объяснение (зачем + как) → интерактив**. Распевочные
упражнения используют детекцию высоты тона (нотный хайвей); дыхательные — таймер/громкость.

## Дальше (план развития, ещё не сделано)
**Новые упражнения (каждое = объяснение + интерактив):**
- Дыхание: «лёгкое перо» (тянуть тихий ровный звук — не дать «перу» упасть), вдох-задержка-счёт.
- Вокал: staccato (Ha-ha-ha в ритм), арпеджио (трезвучие), interval jumps (поймай интервал), messa di voce (pp→ff→pp, двухосевой дисплей pitch×громкость), контроль вибрато, sustain с визуализацией ровности.
- Артикуляция/дикция: скороговорки на ритм.

**Система:**
- Маршрут обучения с уровнями L1–L4, разблокировка по точности (80% за 3 сессии).
- «Почему ошибка» — короткий совет после упражнения (flat/sharp/зажим → дыхание/резонанс). **Дифференциатор №1 vs рынок.**
- Дашборд роста диапазона во времени, ежедневные челленджи, бейджи.
- Аккаунты + облачная синхронизация (сейчас localStorage).
- Song-mode (петь по мелодии песни) + аккомпанемент/метроном (Tone.js).
- Каждое упражнение: экран-теория (что, зачем, как правильно) + видео/гиф-демо.
- **Интеграция с обучением «Прояви»** (вокальная онлайн-школа, `D:\proyavlennost\`): маршрут уроков, мостик в воронку, кабинет ученика/педагога.
