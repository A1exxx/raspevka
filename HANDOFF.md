# HANDOFF — «Распевка» (вокальный тренажёр)
_Обновлено 2026-06-04. Полный контекст для продолжения в новой сессии и для отката._

## Что это
Веб-приложение (PWA) для тренировки вокала: игровые распевки с детекцией высоты тона
в реальном времени + дыхание + определение типа голоса + прогресс. Идея — как Vocalista
(«Sing on Mah»), но web-first, русскоязычное, с упором на корректную методику.

## Где и как

## Что есть сейчас (функционал)
**Главный экран:** чип стрика (огонь), крупная карточка «Полная распевка», бенто-плитки
(Распевайся / Мой голос / Прогресс), распевки — крупными карточками с музыкальными иконками,
дыхание/артикуляция — тонкими строками.

**Тип голоса** (`voice.js` + `voice-types.js`): выбрать (бас…сопрано) или определить —
спеть низ/верх (усреднение + проверка стабильности) → классификация по диапазону.
Мини-клавиатура с закрашенным диапазоном (`illustrations.miniKeyboard`).

**Распевки** (нотный хайвей, `game.js` + `note-highway.js`):
- Перед каждой — **аккорд тоники** + образец мелодии (опора, чтобы попасть).
- **Повтор с транспозицией** (`exercises.transposePlan`): вверх по полутону до верха диапазона и вниз. Счётчик «повтор k/N», агрегированный счёт.
- **Звук-поводырь**: тихо подыгрывает нужную ноту. Тембр на выбор — **Пиано / Гитара / Мягкий** (`reference-tone.playTone(timbre)`, ноты звенят и затухают естественно). Режимы: prehear (без наушников — тон ПЕРЕД нотой, не течёт в микрофон) / continuous (с наушниками). Тумблер «Наушники».
- **Хвост за шариком голоса** — линия истории показывает направление (вверх/вниз).
- **Tier 2:** живая подсказка ↑выше/↓ниже/в точку; разбор «почему» после (тенденция занижения/завышения + слабые верхние ноты).
- **Уровни сложности** (темп) + все настройки прямо в упражнении (значок ⚙) + на экранах объяснения/итога.
- Упражнения: Мычание (М, I-II-III-II-I), Губной тренаж (brrr/Р, 5 нот+квинта), Удержание, Гамма «Ма-Мэ», Беглость «Ма», Октавный скачок.

**Полная распевка** (`session.js`): дыхание длинные с/ш → короткий с+5ш → 15с/15ш → мычание М → губной тренаж. Ритм-распевки через `rhythm.js` (метроном + дрон-подложка, без питча).

**Дыхание** (`breathing.js`): box 4-4-4-4 (дышащий круг), дыхание животом 4-6, долгий выдох «с-с-с» (замер + рекорд). **Картинка-человек** дыхания животом — ИИ-иллюстрация (слева спущенный, справа надутый живот), `belly-diagram.js` + `src/ui/belly-breathing.webp`.

**Распевайся** (`freesing.js`): свободное мычание — вертикальная нотная шкала + шарик + хвост, центы, индикатор уровня, **регулятор чувствительности микрофона** (объединяет тюнер). 

**Микрофон** (`mic.js`): getUserMedia (autoGainControl on) + **GainNode (усиление входа) → analyser** + pitchy. Чувствительность Низкая/Средняя/Высокая (×1.5/3/5.5), **авто-дефолт по устройству** (телефон→высокая), ручное переключение в «Распевайся», порог clarity 0.85 + гейт громкости 0.0025 (ловит тихий голос). Подсказка «не слышу» — только при активной ноте и >5с тишины.

**Прогресс** (`progress-dash.js` + `state/progress.js`): стрик, сессии, рекорд выдоха, календарь 14 дней, бары точности, рост диапазона (история диапазона). Всё в localStorage.

**Надёжность:** пауза прохода при сворачивании экрана (visibilitychange), отменяемые тоны/таймеры, скролл, reduced-motion, дисклеймер здоровья голоса (один раз).

## Что любит/не любит пользователь
- Любит: красиво, стильно, минималистично-но-интересно; распевки крупно, дыхание — компактно; понятные картинки-люди (не схемы).
- Не любит: схемы вместо людей; короткий «тук» вместо звенящей ноты; назойливые подсказки; чтобы микрофон «не слышал».
- Планировать перед крупными правками; сохранять рабочие версии тегами.
