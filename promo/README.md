# Промо-ролики «Распевка»

Вертикальные ролики 1080×1920 (Reels / TikTok / Shorts), **без звука** —
музыку накладывать из библиотеки площадки при заливке (правило проекта:
не синтезировать звук для рекламы). Бренд: тёмная сцена, teal-свечение.

## Готовые ролики (разные углы — не клоны, чтобы не ловить «дубли контента»)

| Файл | Угол / хук | Визуальный герой |
|------|-----------|------------------|
| `raspevka-promo-15s.mp4` | «Поёшь мимо нот? Теперь это видно» | нотный хайвей + след голоса |
| `raspevka-voice-14s.mp4` | «Какой у тебя голос? За 1 минуту» | клавиатура диапазона → «Баритон» |
| `raspevka-streak-14s.mp4` | «Бросаешь через неделю? Со стриком — нет» | кольцо прогресса + 🔥 стрик + ★ |
| `raspevka-5min-13s.mp4` | «Нет времени на вокал? 5 минут в день» | попадание в зелёную зону + таймер |

Все ведут на финал-бренд «Распевка» (логотип-эквалайзер).

## Исходники
`r2-voice.html` · `r3-streak.html` · `r4-fivemin.html` · `index.html` (ролик 1).
GSAP — локально (`gsap.min.js`), без CDN (стабильный рендер).

## Перерендерить
```
node tools/render-promo.mjs promo/<файл>.html promo/<выход>.mp4 24
```
Движок: Playwright скрабит paused-GSAP timeline покадрово → ffmpeg (H.264, yuv420p, -an).
Важно: seek через `tl.time(t)` в блок-теле (иначе Playwright виснет на сериализации timeline).

## Заливка (помни про политику площадок)
- Музыка — из библиотеки Reels/TikTok при загрузке (не из видеофайла).
- Каждый ролик = свой угол + своё оформление; не лить все в один день — разносить.
- AI-disclosure при необходимости (проверять актуальные правила площадки перед серией).
